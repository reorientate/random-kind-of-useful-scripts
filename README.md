# random-kind-of-useful-scripts
these are just scripts i have that i think might be useful to some people

all of these scripts, like akill or bls, are ment to be PATH executables, you put the scripts in your PATH

Usually, ```/usr/local/bin``` is one of the default PATHs for Unix[like]

## a bit of documentation on these scripts

### akill

Find and kill processes very easily by searching for a keyword or part of the name

```akill <name>```
